# Ultimate Task Manager / To-Do App

## Repository Name
final-web-project

## Project Description
A fully interactive Task Manager web application that allows users to add, complete, delete, and filter tasks. Tasks persist in the browser using localStorage, making it practical, functional, and user-friendly.

## Project Status
⚡ Project planning phase — initial repository setup. Code will be added step by step.

## Features (Planned)
- Add new tasks with title and optional description
- Mark tasks as complete / incomplete
- Delete tasks individually
- Filter tasks: All / Active / Completed
- Persist tasks using localStorage
- Responsive design for mobile, tablet, and desktop
- Polished UI with intuitive interactions

## Technologies Used (Planned)
- HTML5 (semantic structure)
- CSS3 (Flexbox, Grid, responsive design)
- JavaScript (ES6+, DOM manipulation, events, arrays/objects)
- localStorage for data persistence
