# Quest 1 – Project Plan

## Project Title
Ultimate Task Manager / To-Do App

## Repository Name
final-web-project

## Project Description
A fully interactive Task Manager web application that allows users to add, complete, delete, and filter tasks. All tasks persist in the browser using localStorage, making it a practical, functional, and user-friendly productivity tool.

## Features
- Add new tasks with title and optional description
- Mark tasks as complete / incomplete
- Delete tasks individually
- Filter tasks: All / Active / Completed
- Persist tasks using localStorage
- Responsive design for mobile, tablet, and desktop
- Polished UI with intuitive interactions

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (Flexbox, Grid, responsive design)
- JavaScript (ES6+, DOM manipulation, events, arrays/objects)
- localStorage for data persistence

## Why This Project
- Demonstrates strong JavaScript skills and interactivity
- Shows understanding of responsive design and UX
- Fully aligned with grading rubric requirements
- Small enough for step-by-step mastery, but impressive when polished

## Expected Learning Outcomes
- Master DOM manipulation and event handling
- Practice working with arrays and objects in JavaScript
- Implement localStorage for data persistence
- Build a fully responsive, polished web application
- Organize code professionally across HTML, CSS, and JS files
